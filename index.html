NotifyMe

Notifyme manages and transports notifications across the internet to wherever you want them.

It's easy!

Just install the core components and you're all set.

It's flexible!

You can structure your notifications by source or category in resources, so you get only the notifications you want right now.

It's extensible!

NotifyMe is practically a python library, so implementing your own components is really easy. and it will get even better in the future.

It's free! (as in freedom)

NotifyMe is licenced under the GNU AGPLv3. This means you can use it and redistribute it, as long as you include a copy of the GNU AGPLv3.

Core components

NotifyMe is split in four core components:

  • collector and emitter
  • publisher and subscriber
In the current implementation, collector and publisher run on the same machine.

Here's how it works

  1. the emitter generates a notification. This could, for example be your IRC client.
  2. the emitter forwards it to the collector.
  3. the collector forwards it to the publisher.
  4. if a subscriber is connected, the publisher sends it there.
  5. the subscriber displays the notification to the user.
